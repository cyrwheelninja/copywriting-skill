## Typical Workflow

1. **Tell Claude what you need**
   ```
   "Write cold email copy for outreach to SaaS founders"
   ```

2. **Claude loads the right prompt**
   - Reads `18-cold-email.md`
   - Asks you for required inputs (target audience, offer, etc.)

3. **Provide the information**
   - Answer Claude's questions
   - Share any customer data you have

4. **Get your copy**
   - Claude generates options using the prompt
   - Runs AI trope check to remove robotic patterns
   - Delivers clean, human-sounding copy

## AI Trope Detection

The skill automatically removes AI writing patterns like:
- "Dive into", "leverage", "robust", "seamless"
- Generic openings like "In today's fast-paced world..."
- Overuse of em dashes
- Corporate buzzwords

You get human-sounding copy, not robotic fluff.

## Frameworks for Quick Tasks

For small tasks that don't need a full prompt, the skill uses copywriting frameworks:
- **PAS** (Problem, Agitate, Solution)
- **AIDA** (Attention, Interest, Desire, Action)
- **BAB** (Before, After, Bridge)
- **FAB** (Features, Advantages, Benefits)
- **PASTOR** (Problem, Amplify, Story, Transformation, Offer, Response)
- **SCQA** (Situation, Complication, Question, Answer)

Just mention the framework:
```
"Write a product description using the PAS framework"
```

## Tips for Best Results

1. **Provide customer data** - Reviews, interviews, surveys make copy stronger
2. **Share voice examples** - Give 2-3 samples of your brand's writing
3. **Be specific** - "Write headlines for B2B SaaS" beats "Write headlines"
4. **Iterate** - Ask for variations, not just one option
5. **Use the editing prompt** - Run final copy through the 7-sweep editor
