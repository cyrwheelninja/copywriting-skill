# Copyhackers AI Prompts Index

Reference guide for selecting the right prompt file based on copywriting need.

## Value Propositions & Headlines

### 03-value-propositions-50-in-10-mins.md
Generate 50 value proposition headlines with subheads using customer review data. Use when you need to articulate what makes a product unique and desirable. Feeds in VOC data and outputs headlines with the exact customer quotes used.

### 06-brand-tagline.md
Create 50 memorable tagline ideas for a brand. Use when developing slogans, taglines, or short brand statements. Works with or without existing brand information by using proven tagline formulas.

### 09-headlines-crossheads.md
Analyze and optimize H2/H3 crossheads on web pages. Use when website subheadings feel generic like "Who we are" or "How it works" and need to be more compelling and benefit-driven.

### 31-pain-point-headlines.md
Generate 50 headlines based on a specific customer pain point. Use when positioning a product against competitors by highlighting what problems it solves that others don't.

### 32-brainstorm-value-proposition.md
Brainstorm unique value propositions for newsletter signups. Use when newsletter opt-in forms have generic CTAs and need more compelling reasons to subscribe.

Write thank you page copy that continues engagement post-conversion. Use when building thank you pages that collect additional data, encourage sharing, or drive secondary actions.

### 10-lead-pages-banners-popups.md

### 14-button-copy-cro.md

### 17-competitor-analysis-browse-bing.md
Analyze your homepage vs competitor homepage and generate optimization recommendations. Use when you need to identify messaging gaps to exploit and differentiate from competitors.

### 26-404-page.md
Write humorous, on-brand 404 error page copy. Use when you want to turn a frustrating user experience into a brand-building moment with personality-driven copy.
